Prerequisites‚Äã

Before diving into humanoid robotics, it's recommended that you have:

  • Basic programming knowledge (Python/C++)
  • Understanding of mathematics (linear algebra, calculus)
  • Familiarity with Linux command line
  • Interest in robotics and AI concepts

Hardware & Software Requirements‚Äã

  • Operating System: Ubuntu 20.04/22.04 or ROS 2 compatible system
  • Development Environment: ROS 2 Humble Hawksbill or later
  • Simulation: Gazebo Garden or Isaac Sim
  • Programming Languages: Python 3.8+, C++17
  • Hardware: Computer with sufficient processing power for simulation (recommended: GPU with CUDA support)
